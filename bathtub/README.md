
### bathtub - 3D Rendering of Lexis Surface - using shiny
Download whole folder!

Open either server.R or ui.R - doesn't matter which. Click on Run app. Not comprehensively tested. 

Also, as a bonus, rglAnimationR is the code for "recording" a series of rotations of the object, the resulting pictures are in /pics, as is the pdf animation.pdf, which plays the animation. The .tex file that produced the animation is also there.
